Why Would a PDF Be Password Protected?

PDFs can have two types of password protection:

  • Open password — you need to enter a password just to view the file.
  • Permissions password — you can view it, but printing, copying, or editing is blocked.

Both are frustrating, especially when you own the document and just want to open it without jumping through hoops every time. The good news: both can be removed for free.

Important: Only remove password protection from PDFs you own or have explicit permission to unlock. Removing protection from someone else's file without permission may be illegal in your country.

Method 2

Use Google Chrome's Print to PDF

This clever trick works if you know the password (you just want to save a copy without it).

1

Open the password-protected PDF in Chrome and enter the password

2

Press Ctrl+P (Windows) or Cmd+P (Mac) to open Print

3

Change the destination to "Save as PDF"

4

Click Save — the new file has no password

Works great for removing open passwords. However, it may not always strip editing/printing restrictions.

Method 3

Use Adobe Acrobat Reader (Free Version)

If you already have Adobe Acrobat Reader (the free one — not Pro), you can remove the password after opening the file.

1

Open the PDF in Adobe Acrobat Reader and enter the password

2

Go to File → Properties → Security

3

Change the Security Method to No Security

4

Save the file — it's now password-free

Note: this requires the full Acrobat (not just Reader). Reader only lets you view, not edit security settings.

Method 4

Upload to Google Drive

A surprisingly effective workaround that's built into every Google account.

1

Upload your PDF to Google Drive

2

Right-click the file and choose Open with → Google Docs

3

Once open, go to File → Download → PDF Document

4

The downloaded file will have no password

Downside: your file does get uploaded to Google's servers. If your PDF is confidential, use Method 1 instead.

Method 5

Use Preview on Mac

Mac users have a built-in tool that makes this almost effortless.

1

Open the PDF in Preview (default PDF viewer on Mac)

2

Enter the password to open the file

3

Go to File → Export as PDF

4

Save the new file — it won't have a password

Clean, fast, no internet needed. One of the best options for Mac users.
